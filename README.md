- Taliwind CSS

### Librerías
- clsx

## Configuración de compilación

### Dependencias necesarias
- Node JS
- npm

### Comandos para correr el proyecto
```bash
// Instalar dependencias necesarias
$ npm install

// Levantar y correr el proyecto en http://localhost:3000 o el puerto que este disponible
$ npm start
```
